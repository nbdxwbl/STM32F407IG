# STM32F407IG
这里存放了嵌入式实践的通关代码和其他有趣的文件，比如电子钢琴

文件适配福州大学嵌入式实践课程，相关设备配置参数如下：

> [!NOTE]
>
> **实验环境：**FS-STM32F407 开发平台 
>
> 1. ST-Link 仿真器 
> 2. RealView MDK5.10 集成开发软件 
> 3. STM32CUBEMX 图形开发软件 
> 4. PC 机  XP、Window7/8/10 (32/64bit)
> 5. 串口调试工具 
> 6. 串口交叉线  

**电子钢琴文件**可实现在屏幕上显示钢琴的图片，触摸之则会发出相应声音，以实现电子钢琴的功能

**嵌入式通关文件**则是比较综合的文件，包含的功能有：
利用操作系统实现多进程并行，使用全局变量flag，初始为0，按下k3，flag加一，按下k4，flag清零。

当flag=0时，无事发生

当flag=1时，D1灯闪烁，蜂鸣器奏乐

当flag=2时，D3闪烁，同事舵机转动

当flag=3时，蜂鸣器奏乐，串口每隔1s打印一次” liujia “

当flag=4时，气体传感器和管光敏传感器工作，并向串口调节器输出数值，当出现异常时蜂蜜蜂鸣器报警
